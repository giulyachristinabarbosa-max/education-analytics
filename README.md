## Education Analytics: Análise de Dados Escolares

Projeto de Data Analytics focado em identificar padrões de evasão (Churn) e projetar receitas para instituições de ensino. Utiliza dados fictícios para simular cenários reais de gestão acadêmica.

##  Objetivos
- **ETL:** Importação e tratamento de dados brutos (CSV) com limpeza de formatação.
- **Análise de Risco:** Algoritmo para identificar alunos com frequência abaixo de 60% (risco de cancelamento).
- **KPIs Financeiros:** Cálculo automatizado de receita recorrente mensal (MRR).

##  Tecnologias
- **Python:** Script de automação e lógica de análise de dados.
- **SQL:** Consultas estruturadas para banco de dados.
- **Excel/Sheets:** Manipulação da base de dados original.

##  Como executar
1. Clone este repositório.
2. Certifique-se de ter o Python instalado.
3. Execute: `python analise_dados.py`
